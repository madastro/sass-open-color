Open color is a color scheme for UI design. You can use it for font, background, border etc. It contains a gray and twelve colors - {% for color in site.colors %} {% endfor %}. Open color is provided as CSS, SCSS, LESS, Stylus, Adobe library, Photoshop/Illustrator swatches and Sketch palette.

* All the color is tested on deuteranopia and protanopia mode. The process will soon be published in the article.
